Unit Conversion Class / User-Control Maker Tool:

 

This program will create custom user-controls and classes from the unit conversions listed in the left hand list box.

 

 

First select a unit conversion group from the program’s left hand listbox. Then click the units desired for your custom control or class within the top-right listView box. These units will be copied into the bottom listView box as you click upon them. Next, you can rename the control or class symbols that will be displayed in your user control or be used to label your class property headings. Last, click the ‘Make UnitConrtol.ctl’ or ‘Make UnitClass.cls’ button. In the file-save dialog that pops up, select a destination filename and directory for your user-control or class.

 

Two user-control versions to choose from: Narrow / Wide_Beta (option button select).

 

The purpose for using such unit conversion controls or classes in an application is to allow the user to have much flexibility over the choice of input and output unit selections while allowing the programmer to always work with a mathematically consistent set of units such as that may conform to some specific mathematical formula, say: F=ma. The programmer can always work in units of Newton’s = kilograms * meter/sec^2 and never have to risk making a unit conversion error!  The user can work with whatever units they please, say pounds-force, pounds-mass and feet/sec^2. The programmer never has to perform unit conversions to get the right answer – a very safe and handy capability.

 

To use the user-control in your application, copy and paste both the user-control.ctl file and the required support class UnitElement.cls file to your application directory. Then right-click in your Project window and select Add > User Control. In the dialog box that pops up choose the 'Existing' tab and select the user-control.ctl file. You will see the user-control button appear in your toolbox. See the example LoadCell application to look at some user controls that were made with the Unit Conversion Class / User-Control Maker Tool.

 

The unit conversion classes that can be made with the Unit Conversion Class / User-Control Maker Tool are very similar to the user controls but they have no user interface. See the Simple Area Calculator example program to look at how to use unit conversion classes that were made with the Unit Conversion Class / User-Control Maker Tool.
